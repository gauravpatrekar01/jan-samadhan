📱 Breakpoint Information

Current responsive breakpoints applied:
📱 Mobile (< 768px)
  • Charts: Single column
  • Map: 300px min height
  • Heatmap: 2 columns
  • Fonts: Smaller
💻 Tablet (768px - 1024px)
  • Charts: 2 columns
  • Map: 350px min height
  • Heatmap: Auto columns
  • Fonts: Medium
🖥️ Desktop (1024px - 1200px)
  • Charts: 2-3 columns
  • Map: 400px min height
  • Heatmap: 160px min width
  • Fonts: Normal
🖥️ Large (> 1200px)
  • Charts: 3+ columns
  • Map: 450px min height
  • Heatmap: 180px min width
  • Fonts: Larger
