JanSamadhan FAQ

Quick answers to common questions

What is JanSamadhan? +

A smart grievance platform to report, track, and resolve public issues efficiently.

How do I submit a complaint? +

Login → Register Complaint → Fill details → Upload media → Submit.

Who resolves complaints? +

Government officers and verified NGOs handle grievances.

Is my data secure? +

Yes. Sensitive data is protected with encryption and strict role-based access.

Can NGOs handle complaints? +

Yes, only verified NGOs approved by admin can handle complaints.

What if complaint is not resolved? +

The system auto-escalates complaints if SLA is breached.
